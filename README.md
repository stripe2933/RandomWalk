## How to build
Required dependencies: [glfw3](https://github.com/glfw/glfw), [glm](https://github.com/Groovounet/glm), [glew](https://github.com/nigels-com/glew), [fmt](https://github.com/fmtlib/fmt)

```shell
mkdir build
cd build
cmake ..
cd ../
cmake --build build --config release
```
